# Editor de Propuestas CLA - ESS

Aplicación desarrollada para la Coordinación Nacional de la Estrategia Sembremos Seguridad.

## Objetivo

Convertir automáticamente las propuestas de líneas de acción generadas en PDF a una versión editable para procesos de validación, revisión y mejora.

## Funcionalidades

* Carga de PDF de propuesta.
* Reconstrucción automática del documento.
* Generación de PDF editable.
* Observaciones por:

  * Línea de acción.
  * Acción estratégica.
  * Indicador.
  * Meta y unidad.
  * Líder estratégico.
* Resultado de revisión.
* Descarga de PDF revisado.
